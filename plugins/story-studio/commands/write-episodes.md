# Command: write-episodes

คำสั่งนี้ใช้สำหรับสร้างเนื้อหาบทภาพยนตร์สั้นเป็นชุด (Batch) ตามจังหวะที่กำหนด

## การใช้งาน
`/story-studio:write-episodes --range <start>-<end>`

## ขั้นตอนการทำงาน
1. **ดึงบริบท**: ให้ **Context Manager** สรุปบริบทล่าสุดจาก `context.json`
2. **สร้างเนื้อหา**: ให้ **Lead Writer** สร้างเนื้อหาตามช่วงตอนที่กำหนด (เช่น 1-5, 6-25, ฯลฯ)
3. **ตรวจสอบคุณภาพ**: ให้ **Editor** ตรวจสอบความถูกต้อง ตรรกะ และการจัดรูปแบบ
4. **นับคำ**: ให้ **word-counter skill** นับและตรวจสอบ word count แต่ละตอน พร้อมแสดง badge ✅/⚠️
5. **อัปเดตบริบท**: ให้ **Context Manager** บันทึกข้อมูลใหม่รวมถึง `word_count_log` หลังจบแต่ละตอน
6. **Auto-sync Miro** (ถ้ามี `miro_board_ids` ใน context.json): อัปเดต Storyboard และ Word Count Tracker
7. **Auto-sync Notion** (ถ้ามี Notion database IDs): Push ตอนใหม่ขึ้น Episodes database
8. **ส่งมอบ**: ส่งเนื้อหาที่ผ่านการตรวจสอบแล้วให้ผู้ใช้และรอคำสั่ง "ดำเนินต่อ"

## ข้อกำหนดจำนวนคำ
- **ตอนที่ 1-5**: 800-1,000 คำต่อตอน
- **ตอนที่ 6-25**: 650-900 คำต่อตอน
- **ตอนที่ 26-40**: ไม่เกิน 800 คำต่อตอน
- **ตอนที่ 41-60**: ไม่เกิน 700 คำต่อตอน
- **ท้ายแต่ละตอน**: ต้องมี 【จุดดึงดูดตอนหน้า】 (Hook) — ไม่นับรวมใน word count
