# Command: start-project

คำสั่งนี้ใช้สำหรับเริ่มต้นโปรเจกต์การเขียนบทภาพยนตร์สั้นใหม่ โดยจะรวบรวมข้อมูลเบื้องต้นและสร้างไฟล์บริบทพื้นฐาน

## การใช้งาน
`/story-studio:start-project`

## ขั้นตอนการทำงาน
1. **รับข้อมูล**: รับนวนิยายหรือข้อความดั้งเดิมจากผู้ใช้
2. **ยืนยันรายละเอียด**: ยืนยันจำนวนตอนทั้งหมด (40-60 ตอน) และความต้องการเกี่ยวกับมุมมองเฉพาะ
3. **วิเคราะห์เบื้องต้น**: ให้ **Lead Writer** วิเคราะห์พล็อตและตัวละคร
4. **สร้างบริบท**: ให้ **Context Manager** สร้างไฟล์ `context.json` และบันทึกข้อมูลเบื้องต้น
5. **สร้างชื่อเรื่อง**: ให้ **Marketing Specialist** สร้างชื่อเรื่อง 4-5 คำให้ผู้ใช้เลือก
6. **ส่งมอบ**: สรุปแผนการทำงานและรอคำสั่ง "ดำเนินต่อ" จากผู้ใช้
