# Command: update-context

คำสั่งนี้ใช้สำหรับอัปเดตข้อมูลบริบท คีย์เวิร์ด และสถานะตัวละครในไฟล์ `context.json`

## การใช้งาน
`/story-studio:update-context --data <json_data>`

## ขั้นตอนการทำงาน
1. **รับข้อมูล**: รับข้อมูลใหม่จากเอเจนต์หรือผู้ใช้
2. **ตรวจสอบความสอดคล้อง**: ให้ **Context Manager** ตรวจสอบว่าข้อมูลใหม่ไม่ขัดแย้งกับข้อมูลเดิม
3. **อัปเดตไฟล์**: บันทึกข้อมูลใหม่ลงใน `context.json`
4. **รายงาน**: สรุปการเปลี่ยนแปลงที่เกิดขึ้นให้ผู้ใช้ทราบ
5. **แจ้งเตือน**: หากพบความไม่สอดคล้อง ให้แจ้งเตือนเอเจนต์อื่นและผู้ใช้ทันที
