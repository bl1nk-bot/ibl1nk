# Command: edit-content

คำสั่งนี้ใช้สำหรับแก้ไขและตรวจสอบบทความอย่างมืออาชีพ รวมถึงการเปลี่ยนชื่อคน/ชื่อสถานที่จำนวนมาก

## การใช้งาน
`/story-studio:edit-content --operation <formatting|spellcheck|logic|replace_names> --data <content>`

## ขั้นตอนการทำงาน
1. **รับข้อมูล**: รับเนื้อหาบทความที่ต้องการแก้ไข
2. **เลือกฟังก์ชัน**: เลือกฟังก์ชันการแก้ไขและตรวจสอบที่ต้องการ
3. **ตรวจสอบคุณภาพ**: ให้ **Editor** ตรวจสอบความถูกต้อง ตรรกะ และการจัดรูปแบบ
4. **เปลี่ยนชื่อ**: หากมีการเปลี่ยนชื่อ ให้ดำเนินการเปลี่ยนชื่อคน/ชื่อสถานที่จำนวนมากและสรุปเป็นตาราง
5. **ส่งมอบ**: ส่งบทความที่ได้รับการแก้ไขและตรวจสอบอย่างมืออาชีพให้ผู้ใช้
