# Article Editor Skill

สกิลนี้ช่วยในการตรวจสอบและแก้ไขบทความอย่างมืออาชีพ เพื่อให้บทความมีรูปแบบที่สวยงาม ถูกต้องตามหลักไวยากรณ์ มีตรรกะที่สมเหตุสมผล และมีความสม่ำเสมอในการใช้ชื่อเฉพาะ

## Capabilities

- **Professional Review**: ตรวจสอบบทความอย่างละเอียดเพื่อให้แน่ใจว่ามีคุณภาพสูง
- **Formatting**: จัดรูปแบบที่สวยงามและสม่ำเสมอเพื่อเพิ่มความน่าอ่าน
- **Grammar and Spelling Correction**: ระบุและแก้ไขข้อผิดพลาดทางไวยากรณ์และคำสะกดผิด
- **Logical Flow Check**: วิเคราะห์บทความเพื่อหาความไม่สอดคล้องกันของตรรกะและเสนอแนะการปรับปรุง
- **Mass Naming Replacement**: ช่วยในการเปลี่ยนชื่อตัวละครหรือสถานที่จำนวนมากเพื่อให้แน่ใจว่ามีความสม่ำเสมอ

## Usage

1. **Provide Article**: ส่งข้อความบทความที่ต้องการแก้ไข
2. **Select Operations**: เลือกฟังก์ชันการแก้ไขและตรวจสอบที่ต้องการ (เช่น การจัดรูปแบบ, การตรวจสอบคำผิด, การตรวจสอบตรรกะ, การเปลี่ยนชื่อ)
3. **Review Suggestions**: สกิลจะให้คำแนะนำและเน้นจุดที่ควรปรับปรุง
4. **Apply Changes**: เลือกใช้การเปลี่ยนแปลงที่แนะนำโดยอัตโนมัติหรือตรวจสอบด้วยตนเอง
5. **Final Output**: รับบทความที่ได้รับการแก้ไขและตรวจสอบอย่างมืออาชีพ
