# Screenplay Adapter

สกิลนี้ให้ความรู้และเวิร์คโฟลว์เฉพาะทางสำหรับการแปลงข้อความเล่าเรื่องให้เป็นรูปแบบบทภาพยนตร์มืออาชีพ โดยแบ่งเป็นตอนๆ พร้อมข้อกำหนดจำนวนคำที่เฉพาะเจาะจง

## Core Workflow

1. **Analyze the Source Text**: อ่านข้อความเล่าเรื่องเพื่อทำความเข้าใจพล็อต ตัวละคร และธีม
2. **Plan Episode Splitting**: กำหนดวิธีการแบ่งเรื่องราวเป็น 1-60 ตอนตามจุดหักเหของเรื่องและจุดดึงดูด (Cliffhangers)
3. **Apply Formatting Rules**: ใช้รูปแบบบทภาพยนตร์มาตรฐาน (Scene Headings, Action, Character, Dialogue)
4. **Manage Word Counts**: ตรวจสอบให้แน่ใจว่าแต่ละตอนเป็นไปตามข้อกำหนดจำนวนคำ (800-1000 คำสำหรับตอนที่ 1-5, 650-900 คำสำหรับตอนที่ 6-60)
5. **Review and Refine**: ตรวจสอบความสม่ำเสมอของเสียงตัวละคร จังหวะ และรูปแบบในทุกตอน

## Key Instructions

### 1. Screenplay Formatting

ปฏิบัติตามรูปแบบบทภาพยนตร์มาตรฐานเสมอ (Sluglines, Action lines, Character names, Dialogue, Parentheticals, Transitions)

### 2. Episode Structuring

แบ่งเรื่องราวเป็นจำนวนตอนที่ต้องการ (สูงสุด 60 ตอน) โดยคำนึงถึงตรรกะการแบ่งตอนและข้อกำหนดจำนวนคำต่อตอน

### 3. Adaptation Principles

- **Show, Don't Tell**: แปลงความคิดภายในเป็นการกระทำที่มองเห็นได้หรือบทสนทนา
- **Visual Storytelling**: เน้นสิ่งที่ผู้ชมเห็นและได้ยิน
- **Character Consistency**: รักษาเสียงที่แตกต่างกันสำหรับตัวละครแต่ละตัว
- **Pacing**: ใช้การแบ่งตอนเพื่อสร้างความตึงเครียดและทำให้ผู้ชมมีส่วนร่วม
