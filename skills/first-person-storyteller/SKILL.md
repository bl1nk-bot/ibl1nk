# First-Person Storyteller

สกิลนี้ช่วยให้ Manus สามารถสร้างเรื่องเล่าจากมุมมองบุคคลที่หนึ่งได้อย่างมีประสิทธิภาพ โดยเน้นการใช้ภาษาพูดที่ชัดเจน สร้างภาพลักษณ์ที่น่าติดตาม และตัดคำอธิบายที่ไม่จำเป็นออกไป เพื่อให้ผู้อ่านรู้สึกเหมือนอยู่ในสถานการณ์จริง นอกจากนี้ยังมีความสามารถในการสร้างจุดดึงดูดความสนใจ (Hook) ที่น่าตื่นเต้นในตอนท้ายของแต่ละบทหรือตอน เพื่อกระตุ้นความอยากรู้อยากเห็นและดึงดูดผู้อ่านให้ติดตามต่อไป

## Core Capabilities

### 1. การเปลี่ยนมุมมอง (Perspective Transformation)
ความสามารถในการแปลงเรื่องเล่าจากมุมมองบุคคลที่สาม (เขา/เธอ) ให้เป็นมุมมองบุคคลที่หนึ่ง (ฉัน) เพื่อสร้างความรู้สึกใกล้ชิดและดื่มด่ำกับเรื่องราวมากขึ้น โดยจะเน้นการใช้สรรพนามและคำกริยาที่เหมาะสมกับมุมมองบุคคลที่หนึ่ง

### 2. สไตล์ภาษา (Linguistic Style)
การใช้ภาษาพูดที่ชัดเจนและมีชีวิตชีวา เพื่อให้เรื่องราวมีความเป็นธรรมชาติและเข้าถึงง่าย พร้อมทั้งสร้างภาพลักษณ์ที่ชัดเจนในจินตนาการของผู้อ่าน โดยจะกำจัดคำอธิบายที่ไม่จำเป็นออกไป เพื่อให้เรื่องราวกระชับและน่าติดตาม

### 3. การตั้งค่าความตื่นเต้น (Hook Setting)
การสร้างจุดดึงดูดความสนใจหรือ 'Hook' ที่ทรงพลังในตอนท้ายของแต่ละส่วนหรือตอน เพื่อกระตุ้นอารมณ์ความรู้สึกของผู้อ่าน สร้างความลึกลับ หรือจุดชนวนความอยากรู้อยากเห็นให้ติดตามเรื่องราวในส่วนถัดไปอย่างต่อเนื่อง

## Usage Instructions

1. **Input**: รับเนื้อหาเดิมหรือโครงเรื่อง
2. **Process**: แปลงเป็นมุมมองบุคคลที่หนึ่ง (ฉัน) โดยใช้ภาษาพูดที่กระชับ
3. **Output**: ส่งมอบเนื้อหาที่เน้นภาพลักษณ์ "เห็นอะไรก็ได้ผลลัพธ์เช่นนั้น" พร้อมระบุ 【จุดดึงดูดตอนหน้า】 ท้ายตอนอย่างชัดเจน
